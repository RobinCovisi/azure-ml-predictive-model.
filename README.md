# azure-ml-predictive-model.
Um desafio para descrever a criação de um lab no Azure Machine Learning.

# Projeto de Machine Learning no Azure

## Objetivo
Criar um modelo preditivo usando Azure Machine Learning.

## Passo a passo
1. Criação do Workspace no Azure ML.
2. Escolha do dataset (ex.: Diabetes).
3. Configuração do experimento com Automated ML.
4. Treinamento e avaliação do modelo.
5. Publicação do endpoint.
6. Teste do endpoint via JSON.

## Tecnologias usadas
- Azure Machine Learning
- GitHub
